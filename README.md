# 💳 Cartão de Crédito Design
Este projeto foi desenvolvido para praticar HTML e CSS, criando um cartão de crédito estático com layout limpo e estilizado, mostrando informações como número, nome e validade do cartão.

## 🛠️ Tecnologias utilizadas
- HTML  
- CSS  

## 🎯 Objetivo
Praticar a criação de layouts e estilização de elementos usando apenas HTML e CSS, desenvolvendo um cartão de crédito virtual estático e visualmente agradável.

## 🌐 Visualizar projeto
https://liadaine.github.io/cartao-credito-design/

## 📚 O que foi praticado
- Estruturação do cartão em HTML  
- Estilização com CSS (cores, gradientes e posicionamento de elementos)  
- Criação de um layout estático com design limpo  
- Organização de elementos como chip, número, nome e validade do cartão
